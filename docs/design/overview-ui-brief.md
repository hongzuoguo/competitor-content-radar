# Overview UI Brief

- **Visual thesis:** 一张“清晨编辑台”式的明亮工作面，以矿物青信号快速指出今天值得研究的内容。
- **Content plan:** 左侧全局导航；顶部运行状态与主操作；连续横向数据带；今日重点列表；内容趋势和任务健康；右侧作品检查器。
- **Interaction thesis:** 重点行在悬停时显露操作；点击后工作区平滑收窄并打开右侧检查器；任务状态变化用短促进度过渡和行内反馈表达。
- **Scope:** Windows 桌面应用的完整产品界面，生产级交互；首轮覆盖首页、博主管理、作品分析、任务记录和设置。
- **References:** Linear 的克制信息密度、Notion 的自然层级、Raycast 的快速操作感。
- **Required states:** 首次使用、无博主、采集中、正常、部分失败、登录过期、AI 额度不足、飞书授权失效、离线、长文案和大量作品。
